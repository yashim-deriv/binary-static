Sebelum menonaktifkan akun, Anda harus melakukan hal berikut:

Tutup kontrak yang masih berjalan

Jika Anda memiliki akun riil Binary, kunjungi Portfolio untuk menutup posisi berjalan.

Jika Anda memiliki akun riil MT5, akses akun tersebut untuk menutup posisi berjalan.

Tarik dana Anda

Jika Anda memiliki akun riil Binary, kunjungi Kasir untuk menarik saldo Anda

Jika Anda memiliki akun riil DMT5, kunjungi Dashboard MT5 untuk menarik dana Anda.
